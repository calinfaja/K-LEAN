## Quick Command Reference

```bash
# System health
healthcheck
k-lean status

# Code reviews
/kln:quick security audit
/kln:multi architecture review
/kln:deep pre-release review

# Knowledge capture
SaveThis "learned that connection pooling improves performance"
FindKnowledge authentication patterns

# Help
/kln:help
```
