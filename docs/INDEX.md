## Quick Command Reference

### Essential Commands

```bash
# Check system health
healthcheck

# Quick code review
/kln:quickReview qwen security audit

# 3-model comparison
/kln:quickCompare check error handling

# Deep audit with tools
/kln:deepAudit pre-release review

# Factory Droid review
/kln:droid qwen3-coder focus on performance

# Get help
/kln:help
```

### Knowledge System

```bash
# Capture knowledge
GoodJob https://example.com/docs focus on authentication
SaveThis learned that connection pooling improves performance

# Search knowledge
FindKnowledge authentication patterns
```
